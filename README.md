# GumroadBest GitHub Pages Mirror

This repository is a static mirror for GumroadBest articles.

Content is generated by `/home/chandravijay/viral_engine.py`.

Publishing flow:

1. Generate or load an article draft.
2. Publish the article to Blogger.
3. Mirror the same article into this repository under `posts/<slug>/index.html`.
4. Rebuild `index.html` and `posts.json`.

Default Pages URL:

`https://cheetahkinglion.github.io/gumroadbest-pages/`
